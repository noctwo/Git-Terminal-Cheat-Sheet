Git-Terminal-Cheat-Sheet

Terminal
cd [path] change current directory to the specified one
cd ~ go back to home from folder
cd - go back to last folder
ls list contents of current directory
ls -lh list contents of directory in human-friendly format
cp [origin] [destination] copies the given file wherever you want to
mv [origin] [destination] moves or renames the given file
pwd get path of the directory you`re currently in
mkdir create a folder > make directory
mkdir -p [name] create a folder and all its parents, if needed
touch [name] creates a file with the given name
file [name] reports the file type
rm [file] removes given file
rm -rf [file] removes a folder and all of its contents
cat [file] prints a files contents
nano [file] opens the nano file editor to edit a file
head [file] prints the first few lines of a given file
tail [file] prints the last few lines of a given file
Git
git init initialise repository
git add * marks all files as "ready to commit"
git commit -m"[message]" adds message to changed files, marks them as "ready to push"
git push origin [branch] uploads changes to given branch
git pull "pull down" changes to your local repo
git status shows tracked / untracked files
git remote add origin [remote URL] connects remote repo with your local repo
git clone [remote URL] clones a remote repo to your local machine
git diff shows made changes
git log--online --decorate --graph --all shows a graph of all made changes
